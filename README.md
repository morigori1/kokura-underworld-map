# Kokura Underworld Map — 小倉組織犯罪史タイムマシン

北九州市小倉を中心に、特定危険指定暴力団 **工藤會**(草野一家 → 工藤連合草野一家 → 工藤會)
を軸とする戦後〜現代の組織犯罪史を、公開報道・判決文・警察白書だけを使って
「上空から、時間をさかのぼって」可視化する OSINT ダッシュボードです。

工藤會は日本で唯一「特定危険指定暴力団」に指定されている組織で、市民への直接攻撃
(元漁協理事射殺・元警察官襲撃・看護師襲撃・歯科医師襲撃 ほか)が裁判で認定された
極めて特異な組織犯罪史を持ちます。本プロジェクトは、それらを衛星タイムライン・
事件タイムライン・判決抜粋・周辺生活情報の 1枚 HTML にまとめます。

姉妹プロジェクト [Compound Time Machine](../SS/)(メコン地域の詐欺コンパウンド OSINT)
の構造を踏襲した小倉版です。

---

## 見る

`index.html` をブラウザでダブルクリックするだけ。サーバーもインストールも不要。

- **インターネット接続が必要**です(衛星タイル・OSM タイル・事件写真を外部から
  読み込むため)。オフラインだと地図が表示されません。地図ライブラリ(Leaflet)は
  `vendor/` に同梱済みなので、CDN がブロックされても UI 自体は読み込めます。
- 共有する場合は **`index.html`・`images/`・`vendor/`** をセットでコピーして
  ください(相対パス参照のため同階層に必要)。
- 文字コードは UTF-8。Windows / Mac / 主要ブラウザで閲覧できます。

操作: スプラッシュ画面で「ガイドツアー」または「自由に探索」を選択。ツアーでは
各拠点へカメラが降下し、衛星が自動再生されます。「◫ 比較」で年代を並べ、
左に事件、右に判決抜粋・証言、ツアーパネルに周辺の街カードが表示されます。

## データを再生成する

派生データ(衛星フレーム・POI・イベント・画像・証言・周辺生活)とダッシュボードは、
ベースとなる `kokura.db` から再生成できます。Python 3 のみ(標準ライブラリだけ、
`pip install` 不要)。

```
python build_all.py          # 全フェーズを実行し index.html を再生成
python build_all.py dash     # index.html だけ再生成(再取得なし)
```

全実行は数分かかり、インターネット接続が必要です(公開 API へアクセス)。各フェーズは
冪等で、再実行すると自前の行を入れ替えます。

実行順(`build_all.py` が自動で処理):

| 順 | スクリプト | 内容 |
|---|---|---|
| 1 | `phase5_poi.py` | POI(関連事件発生地概略・OSM 周辺地物)/ ナレーション / 年代キャプション |
| 2 | `phase4_wayback.py` | Esri Wayback の履歴衛星フレーム(工藤會本部跡 2014→現在 ほか) |
| 3 | `phase6_events.py` | 報道・判決ベースのイベント(1998〜現在) |
| 4 | `phase7_images.py` | Wikimedia Commons の地点画像(`images/` に保存) |
| 5 | `phase8_event_images.py` | 各事件のプレビュー画像 URL(og:image) |
| 6 | `phase9_testimony.py` | 被害者公開証言 + 判決文抜粋 |
| 7 | `phase10_local_life.py` | 旦過市場・魚町銀天街・堺町など周辺の街スニペット |
| 8 | `dash5.py` | `index.html` を生成 |

## ファイル構成

```
index.html      成果物(これを開く)
kokura.db       SQLite データベース(ベース + 派生データ)
images/         Commons の地点画像(ローカル保存)
vendor/leaflet/ 地図ライブラリ Leaflet 1.9.4(同梱・CDN 非依存)
build_all.py    一括再生成スクリプト
phase4〜10_*.py 派生データ生成スクリプト
dash5.py        ダッシュボード生成スクリプト
legacy/         初期のベースデータ構築スクリプト(下記参照)
```

`legacy/` は将来、`kokura.db` のベースデータを最初に構築したスクリプトを保管する
場所です(SS と同じ構成方針)。

## データ層(主な収録内容)

- 拠点 **複数**(工藤會本部跡・主要事件発生地・関連事務所跡など)
- 観測フットプリント(OSM 建物 + 周辺 POI)
- 履歴衛星フレーム(2014〜2026、Esri Wayback)— 本部解体(2019-07)前後のタイムラプス
- イベント(1998〜現在、出典付き)— 市民襲撃4事件・頂上作戦・本部解体・各判決
- 系譜(`chronicle`)— 草野一家初代結成 → 工藤組合流 → 工藤連合草野一家 → 工藤會
- 訴訟経過(`prosecution`)— 福岡地裁 2021-08-24 一審 / 福岡高裁 控訴審 / 上告審
- 証言 — 法廷意見陳述・判決文公開抜粋(被害者は判決文上の役職表記に統一)
- 周辺の今 — 旦過市場(2022 火災)・魚町銀天街・堺町歓楽街の現況

## 出典・データソース

- **衛星**: Esri World Imagery / Wayback
- **地図 / 建物**: OpenStreetMap contributors
- **地点画像**: Wikimedia Commons
- **イベント / 証言 / 判決抜粋 / 周辺情報**:
  - 西日本新聞・毎日新聞・朝日新聞・読売新聞・産経新聞・日経新聞
  - NHK・RKB 毎日放送・FBS 福岡放送・KBC 九州朝日放送
  - 共同通信・時事通信
  - 福岡地方裁判所判決(2021-08-24)・福岡高等裁判所判決(2024)
  - 警察白書(国家公安委員会・警察庁)・福岡県警察本部発表
  - 国正武重『工藤會壊滅作戦』ほか書籍報道

個々の出典 URL はダッシュボード内および DB の `source` テーブルに記録しています。

## 公開する

静的ファイルだけで動くので、どのホスティングでも置けば公開できます。

**最小セット**(これだけで公開可能、推奨):
```
index.html
images/
vendor/
.nojekyll
LICENSE       (任意)
README.md     (任意)
```
`kokura.db`・`phase*.py`・`dash5.py`・`build_all.py`・`legacy/` は閲覧には不要
(手元での再生成用)。

### ホスティング例

- **Netlify Drop** — https://app.netlify.com/drop に上記フォルダをドラッグ → 即座に URL 発行
- **GitHub Pages** — 新規リポジトリに push → Settings → Pages → Branch: `main` / root
- **Cloudflare Pages / Vercel** — GitHub 連携で同様

## 注意・倫理

- 本プロジェクトは **防御・教育目的の OSINT 可視化**であり、すべて公開情報のみを用います。
- 座標は次の二段階方針:
  - **公的な建物・既に解体/閉鎖された施設**(工藤會本部跡・旦過市場など)は具体的座標
  - **個人の住居・職場が舞台の襲撃事件**は **町丁目重心** に丸め、`uncertainty_m` で
    精度を明示。被害者特定を助長する番地レベル座標は載せません。
- 衛星フレームは z15 タイルのハッシュ差分で抽出しており、雲・季節差も含まれ得ます。
- イベント・ナレーション・周辺スニペットは公開報道に基づく編集テキストで、
  現地確認ではありません。
- 被害者は判決文や主要報道で公表されている職業・役職のみで参照し、氏名は記載しません。
- イベントビューアの写真は出典記事の og:image をリンク表示するもので、著作権は各媒体に
  帰属します(ローカル閲覧用途を前提)。
- 本ダッシュボードは **歴史記録の可視化**であり、現存組織や個人を標的化する目的では
  ありません。
